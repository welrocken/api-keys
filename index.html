Youtube

  1. https://console.developers.google.com/ adresine gidilecek. Açılan sayfa
  2. Hesap bilgileri girilerek giriş yapılacak. Açılan sayfa
  3. Bu sayfada "Proje Oluştur" tuşuna basılacak. Açılan sayfa
  4. Bu sayfada proje adı girildikten sonra, "Oluştur" tuşuna basılarak proje oluşturulacak. Açılan sayfa
  5. Bu sayfada "API'LERİ VE HİZMETLERİ ETKİNLEŞTİR" tuşuna basılacak. Açılan sayfa
  6. Açılan sayfadaki arama çubuğuna "youtube data api v3" yazılacak. Açılan sayfa
  7. Aşağıda gelen kutucuk (Youtube Data API V3) tıklanacak. Açılan sayfa
  8. Bu sayfada "Etkinleştir" tuşuna basılarak, Youtube Data API V3 kullanımı onaylanacak. Açılan sayfa
  9. Bu sayfada "Kimlik Bilgileri Oluştur" tuşuna basılacak. Açılan sayfa
  10. Bu sayfadaki alanlar şöyle doldurulacak:
    1. Youtube Data API V3 seçilecek.
    2. Web sunucusu (örneğin node.js, Tomcat) seçilecek.
    3. Herkese açık veriler seçilecek.
    4. "Hangi kimlik bilgilerine ihtiyacım var?" tuşuna basılacak.
    Açılan sayfa

  11. ÖNEMLİ

    Bu sayfada
    1. İşte API anahtarınız sekmesinde verdiği anahtar saklanacak.
    2. Bitti tuşuna basılacak.
    Açılan sayfa

Twitter

  1. https://twitter.com/login?redirect_after_login=https:/developer.twitter.com/ adresine gidilecek. Açılan sayfa
  2. Hesap bilgileri girilecek. Hesaba girildikten sonra, https://developer.twitter.com/en/apply/user adresine gidilecek. Açılan sayfa
  3. "Continue" tuşuna basılacak. Açılan sayfa
  4. "I am requesting access for my organization" seçeneği seçildikten sonra gerekli bilgiler doldurulacak.
    1. Bu formdaki tüm adımların tamamlanmasının ardından Twitter tarafından bir doğrulama e-postası gelecek. Bu e-posta'daki doğrulama linkine tıklanacak.
    2. Twitter bir süre talebi inceledikten sonra (yaklaşık bir gün), bir onay e-postası gönderecek.
    3. Bu e-posta geldikten sonra, https://developer.twitter.com/en/apps/create adresine gidilecek.
    Açılan sayfa
  5. Açılan sayfadaki bilgiler doldurulacak:
    1. App name, uygulamanın adını belirtecek. Örn. Eczacıbaşı Web Sayfası olabilir.
    2. Web site url, https://www.eczacibasi.com.tr olacak
    Açılan sayfa

  6. ÖNEMLİ

    Açılan sayfada, öncelikle üstteki tablardan "Keys and tokens"'a geçilecek.

    Bu sekmedeki Consumer API Keys başlığının altındaki bilgileri saklayınız.

Instagram

    Instagram API bağlantısı için hesap oluşturma sayfası arızalıdır.

  1. https://www.instagram.com/accounts/login/?next=/developer/register/ adresine gidilecek.
  2. Hesap bilgileri girilerek giriş yapılacak. Açılan sayfa
  3. Sayfada istenen alanlar doldurulacak. Açılan sayfa
  4. Bu sayfada "Register Your Application" tuşuna basılarak, uygulama kaydetme sayfasına gidilecek. Açılan sayfa
  5. Bu sayfada "Register a New Client" tuşuna basılarak, yeni bir hesap api hesabı oluşturma sayfasına gidilecek.
  6. Bu sayfadaki bilgiler doldurulduktan ve hesap oluşturulduktan sonra https://www.instagram.com/developer/clients/manage/ sayfasına gidilecek. Açılan sayfa
  7. Bu sayfadaki "MANAGE" tuşuna basılacak. Açılan sayfa

  8. ÖNEMLİ

    Bu sayfada yer alan "Client ID" ve "Client Secret" saklanacak.
